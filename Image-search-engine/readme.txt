Image Search Engine Website
Overview:
This image search engine website allows users to search for images based on keywords. It utilizes the Unsplash API to fetch relevant images and display them on the webpage. Users can enter search queries in the provided search box and browse through the search results. Additionally, there's a "Show More" button to load more images as the user scrolls through the page.

Features:
Keyword-based image search functionality.
Integration with the Unsplash API to fetch images.
Pagination support for browsing through multiple pages of search results.
Installation:
Clone or download the repository to your local machine.
Open the index.html file in a web browser to access the image search engine website.
Usage:
Enter a keyword in the search box.
Press Enter or click the "Search" button to initiate the search.
Scroll through the search results to view the images.
Click on any image to view it in full size on the Unsplash website.
Click the "Show More" button to load additional images.
